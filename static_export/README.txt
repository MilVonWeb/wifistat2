WiFi Doktor — Statický web
==========================

Obsah složky
------------
  index.html          — hlavní stránka webu
  assets/bundle.js    — veškerý JavaScript
  assets/bundle.css   — veškerý CSS
  images/             — obrázky a media soubory
  favicon.svg         — ikona webu
  opengraph.jpg       — náhledový obrázek pro sdílení

Jak nasadit
-----------
1. Nahrát OBSAH této složky do kořenového adresáře hostingu
   (přes FTP, cPanel File Manager, nebo jiného správce souborů)
2. Web je okamžitě funkční — žádný Node.js, PHP ani databáze

Testování lokálně
-----------------
Soubor index.html lze otevřít přímo v prohlížeči (dvojklik).
Web je plně funkční i bez serveru.

Požadavky
---------
- Žádné! Stačí libovolný statický hosting nebo webový server.
- Fonty se načítají z Google Fonts (potřebné internetové připojení).
